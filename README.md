Запуск локально:

```npm start```

Поиск реализован в ```findStreets(request)```.

В функции в цикле выполняется ```indexOf```, поэтому оцениваю сложность как O(N^2).

Для более умного поиска перед запуском для каждой улицы определяется тип (бульвар, проспект и прочие), что позволяет потом искать только по названию и отделять, например, ```Волоколамский проезд``` от ```Волоколамского шоссе```.

Последовательность ввода названия роли не играет, то есть ```улица Красина``` и ```Красина, улица``` дадут одинаковый результат.
